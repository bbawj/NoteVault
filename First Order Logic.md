# First Order Logic
[[Propositional logic]] can only deal with a finite number of propositions:

T: Tommy is faithful
J: Jimmy is faithful
L: Laika is faithful
$All\ dogs\ are\ faithful\iff T\land J\land L$ 
What if there is an infinite/unknown number of dogs?
